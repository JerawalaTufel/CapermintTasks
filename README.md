# CAPERMINT_TASK

## Description
Overview:

Create a express.js application that serves as the backend for a CAPERMINT_TASK. 
## Getting Started
These instructions will help you set up and run the project on your local machine.

### Prerequisites
- Node.js: Ensure that Node.js is installed on your machine. You can download it from [https://nodejs.org/](https://nodejs.org/)

### Installing
1. Clone the repository to your local machine:
   ```bash
   git clone git@github.com:JerawalaTufel/CapermintTasks.git
2. Navigate to the project directory:
    ```bash
    cd your-project
3. checkout in development branch 
    ```bash
    git checkout main
4. Install the project dependencies using npm:
    ```bash
    npm install
5. make .env file after that provide your database connection(MONGO_URL) and add JWT_KEY key.

6. To start the application, run the following command:
    ```bash
    npm start
7.  We have provide postman collection within the project. You can use it as reference for the API Request and Response.
Import it in your postman for api API examples.
    ``` 
    'CapermintTasks.postman_collection.json'
